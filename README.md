# 📈 DeusJ 基金实时估值查询

一个纯前端的基金实时估值查询工具，单个 HTML 文件，无需后端服务，打开浏览器即可使用。

## ✨ 功能特性

- **实时估值查询** — 输入 6 位基金代码，即刻获取实时估算净值及涨幅
- **热门基金** — 内置热门基金快捷入口（易方达中小盘、易方达蓝筹、招商白酒、华夏能源革新、景顺长城新兴成长、交银新成长、天弘沪深300ETF联接、沪深300ETF）
- **收益估算器** — 支持按持有份额或持有金额两种方式，实时计算今日预估收益、估算总市值
- **自选列表** — 添加关注的基金到自选，每 30 秒自动刷新估值，自选列表支持一键移除
- **真实净值对比** — 同时展示已公布的真实净值（最近 5 期）及日涨幅，与估值对比参考；自选列表中也展示真实净值及是否当日已更新
- **前十大持仓** — 展示基金前十大重仓股票及占净值比，并实时拉取各重仓股当日涨幅和最新价格
- **关联板块标签** — 根据基金名称和行业配置接口自动识别关联板块（如白酒、新能源、沪深300、港股等），分类着色展示
- **查询历史** — 自动记录最近 10 条查询，支持逐条删除或一键清空，历史标签显示涨跌颜色
- **本地存储** — 自选列表和查询历史保存在浏览器 localStorage，刷新页面不丢失
- **响应式设计** — 适配手机和桌面端，表格横向可滚动

## 🚀 使用方法

1. 下载 `fund-valuation-query.html`
2. 用浏览器直接打开即可，无需安装任何依赖
3. 输入 6 位基金代码查询，或点击热门基金标签快速查看

> 也可部署到任意静态托管服务（GitHub Pages、Vercel、Netlify 等）直接使用。

## 📦 技术栈

- **纯 HTML / CSS / JavaScript**，单文件，零依赖
- 数据来源：天天基金公开接口（JSONP）
- 真实净值：天天基金 F10 数据接口
- 股票行情：东方财富行情推送接口
- 行业配置：天天基金 F10 行业配置接口
