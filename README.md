# 📈 基金实时估值查询

一个纯前端的基金实时估值查询工具，无需后端服务，打开浏览器即可使用。

## ✨ 功能特性

- **实时估值查询** — 输入 6 位基金代码，即刻获取实时估算净值及涨幅
- **热门基金** — 内置热门基金快捷查询（易方达蓝筹、招商白酒、沪深300ETF 等）
- **收益估算器** — 输入持有份额或金额，实时计算今日预估收益
- **自选列表** — 添加关注的基金到自选，每 30 秒自动刷新估值
- **真实净值对比** — 同时展示已公布的真实净值及涨幅，与估值对比参考
- **前十大持仓** — 展示基金前十大重仓股票及实时涨幅
- **查询历史** — 自动记录最近 10 条查询，一键再次查看
- **本地存储** — 自选列表和查询历史保存在浏览器 localStorage，刷新页面不丢失
- **响应式设计** — 适配手机和桌面端

## 🚀 使用方法

下载 `fund-valuation-query.html`，用浏览器直接打开即可，无需安装任何依赖。

## 📦 技术栈

- **纯 HTML / CSS / JavaScript**，单文件，零依赖
- 数据来源：天天基金公开接口（JSONP）
- 股票行情：东方财富行情接口

## ⚠️ 免责声明

- 估值数据来源于公开接口，仅供参考，不构成投资建议
- 实际净值以基金公司披露为准，数据可能存在延迟
